# TP Terraform Azure

Ce TP contient la création de deux machines virtuelles sur Azure avec Terraform.

## Contenu du dépôt
- `main.tf` : fichier Terraform principal
- `suivi_avancement.txt` : suivi de ce qui a été fait et appris
- `.gitignore` : pour ignorer les fichiers locaux Terraform

## Difficultés rencontrées
- Gestion des fichiers volumineux du provider Terraform (supprimés pour GitHub)
- Utilisation de SSH keys et token pour GitHub
- Apprentissage du push correct vers GitHub

## Ce que j'ai appris
- Créer et configurer des ressources Azure avec Terraform
- Gérer le versioning Git avec des fichiers volumineux
- Écrire un suivi d’avancement clair
